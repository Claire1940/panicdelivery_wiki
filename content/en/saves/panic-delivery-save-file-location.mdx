To find your **panic delivery save file location**, follow the standard directory path used by most Unreal Engine titles on Windows. You will need to enable "Hidden Items" in your File Explorer view settings to access these folders.

| Folder Level | Path / Folder Name |
| :--- | :--- |
| Drive | C: (Default System Drive) |
| User Directory | Users > [YourUsername] |
| Hidden Folder | AppData > Local |
| Game Folder | PanicDelivery > Saved > SaveGames |

<Callout type="tip" title="Pro Tip">
You can quickly access the AppData folder by pressing `Win + R` on your keyboard, typing `%localappdata%`, and hitting Enter. This bypasses the need to manually navigate through hidden system folders.
</Callout>

<Section icon={Settings} title="Managing and Backing Up Progress">

Regularly backing up your save files is a smart habit, especially during the Early Access phase where patches and hotfixes are frequent. While Steam Cloud saves are reliable, local copies provide an extra layer of security.

<CardGrid cols={2}>
  <Card title="Manual Backup">
    - Copy the entire `SaveGames` folder.
    - Store in a secondary drive or cloud service.
    - Label backups with the current date (e.g., 2026-06-22).
  </Card>
  <Card title="Cloud Syncing">
    - Ensure Steam Cloud is enabled in Properties.
    - Check for "Up to date" status in Library.
    - Avoid interrupting Steam during file syncing.
  </Card>
</CardGrid>

<Callout type="warning" title="Important Warning">
Never modify, delete, or replace your save files while the game is running. Doing so can lead to file corruption or the loss of your character cosmetics and progress. Always exit to the desktop completely before touching your save data.
</Callout>

</Section>

<Section icon={Target} title="Troubleshooting Save Issues">

If you encounter issues where your progress is not saving or the game fails to load your profile, follow these steps to verify your file integrity.

<Steps>
  <Step num="1" title="Verify Game Files">
    Open Steam, right-click Panic Delivery, select Properties > Installed Files, and click "Verify integrity of game files."
  </Step>
  <Step num="2" title="Check Permissions">
    Ensure your user account has Read/Write permissions for the AppData folder.
  </Step>
  <Step num="3" title="Clear Cache">
    Sometimes, clearing the Steam download cache can resolve synchronization conflicts between your local machine and the cloud.
  </Step>
</Steps>

| Issue Type | Recommended Solution |
| :--- | :--- |
| Save not syncing | Check Steam Cloud settings in Properties |
| Corrupted data | Restore from your manual local backup |
| Missing progress | Verify game files via Steam Client |
| Permissions error | Run Steam as Administrator |
