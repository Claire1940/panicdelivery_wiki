<Section icon={Target} title="Step-by-Step Access Guide">

<Steps>
  <Step num="1" title="Open Run Command">
    Press `Win + R` on your keyboard to open the Run dialog box.
  </Step>
  <Step num="2" title="Navigate to AppData">
    Type `%LOCALAPPDATA%` and press Enter. This will open the local application data folder for your user profile.
  </Step>
  <Step num="3" title="Locate Game Folder">
    Look for the folder named `PanicDelivery`. If it is not present, ensure you have launched the game at least once.
  </Step>
  <Step num="4" title="Access Save Data">
    Navigate into `Saved` > `SaveGames`. This is where your career progression and cosmetic unlocks are stored.
  </Step>
</Steps>

<Callout type="warning" title="Data Safety Warning">
Never modify these files while the game is running. Editing your save file can lead to corrupted progress, which may not be recoverable even with Steam Cloud synchronization.
</Callout>
