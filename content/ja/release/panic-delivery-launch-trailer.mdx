<Section icon={Map} title="ローンチ概要とゲームプレイの基本">

**『Panic Delivery』ローンチトレーラー**は、運び屋が「使い捨て」とされる過酷な環境を映し出しています。2026年5月13日のリリース以来、プレイヤーはモンスターが蔓延る危険地帯で配達任務を遂行することになります。

**動画の注目ポイント：**
- 最大4人のカオスな協力型配達メカニズム。
- 共有の99ライフシステムの紹介。
- モンスターが徘徊する様々なディストピアマップ環境の先行公開。

<YouTubeEmbed videoId="HKjn7oej6MM" title="Panic Delivery - Early Access Launch Trailer" />

本作で成功するには、スピードだけでなくチームの厳格な連携が求められます。運び屋が倒れるとチームのライフが1つ減り、0になるとランがリセットされます。ただし、個人で獲得したコスメティックの進行状況は保持されます。

| 特徴 | 説明 |
| :--- | :--- |
| **プレイヤー数** | 最大4人のオンライン協力プレイ |
| **ライフシステム** | チーム共有の99ライフ |
| **ゲーム設定** | モンスターが徘徊するディストピアゾーン |
| **進行状況** | ショップでのアップグレードと装飾アイテムのアンロック |

<Callout type="tip">
一人で危険に飛び込まないでください。チームが一丸となって動き、常に荷物運び役を保護するチームが評価されます。
</Callout>

</Section>

<Section icon={Target} title="ミッションタイプと配達のロジック">

目的を理解することは生存への第一歩です。アーリーアクセス版では3つの異なるミッションタイプが用意されており、それぞれランダムな目標が設定されるため、ルートが予測不可能になっています。

<Steps>
  <Step num="1" title="ミッションタイプの確認">
    開始前に、ミッションが「スタンダード」「リカバリー」「密輸」のどれかを確認してください。それぞれ移動や防御の戦略が異なります。
  </Step>
  <Step num="2" title="パッケージ戦略の選択">
    「小」「中」「大」のパッケージは移動速度に影響します。大きな荷物は重く、安全に運ぶにはチームのサポートが不可欠です。
  </Step>
  <Step num="3" title="任務遂行と脱出">
    目標を達成し、脱出ポイントへ向かいましょう。チームの共有ライフを無駄にしないよう、常に残数に注意してください。
  </Step>
</Steps>

| ミッションタイプ | 目標の焦点 | リスクレベル |
| :--- | :--- | :--- |
| **スタンダード** | 地点間配送 | 中 |
| **リカバリー** | 紛失した荷物の捜索と回収 | 高 |
| **密輸** | 違法品の保護輸送 | 極めて高い |

<Callout type="warning" title="目標に関する注意">
目標はランダムに変化するため、途中で脱出ルートが変わる可能性があります。ミッションマーカーだけでなく、周囲の状況にも常に注意を払ってください。
</Callout>

</Section>
