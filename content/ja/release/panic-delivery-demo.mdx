<Section icon={Target} title="ミッションタイプと荷物のロジック">

任務を理解することが、成功への第一歩です。各ミッションタイプには、荷物を確実に目的地へ届けるための異なるアプローチが必要です。

| ミッションタイプ | 主な目標 | プレイヤーの戦略 |
| :--- | :--- | :--- |
| スタンダード | 指定された荷物を届ける | 何としても運び役を守る |
| 回収 | 紛失した貨物を見つけて確保する | 追跡デバイスを使用してアイテムの位置を特定する |
| 密輸 | 不正品を安全に運ぶ | トラップを避けるために移動を調整する |

<Callout type="warning" title="貨物のリスク">
大きな荷物は運び役の移動速度を大幅に低下させます。運び役が高リスクゾーンに入る前に、チームが道を確保し、脅威を排除してください。
</Callout>

</Section>

<Section icon={Shield} title="生存とショップアイテム">

ショップアイテムを効果的に活用すれば、失敗しそうなランも成功する配達に変えることができます。Speetchy（スピーチー）を賢く使うことが、進行を維持する鍵です。

| アイテムカテゴリ | 最適な使用例 | ティア評価 |
| :--- | :--- | :--- |
| 追跡デバイス | 回収ミッション | S-Tier |
| バール | 箱の破壊・防御 | S-Tier |
| モンスター忌避剤 | 緊急時の脱出ルート確保 | A-Tier |
| 移動用ツール | 偵察・位置調整 | A-Tier |
