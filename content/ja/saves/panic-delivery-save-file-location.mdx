<Section icon={Map} title="セーブデータの場所">

PCプレイヤーにとって、ローカルファイルを管理することは、この協力型ホラーコメディゲームの進捗を維持するための重要な要素です。本作はSteamを通じて配信されているため、セーブデータはローカルで処理され、クラウドに同期されます。

<YouTubeEmbed videoId="HKjn7oej6MM" title="Panic Delivery アーリーアクセスローンチトレーラー" />

**Panic Deliveryのセーブデータ保存場所**を見つけるには、Windows上のほとんどのUnreal Engineタイトルで使用される標準のディレクトリパスに従ってください。これらのフォルダにアクセスするには、ファイルエクスプローラーの表示設定で「隠しファイル」を表示する必要があります。

| フォルダ階層 | パス / フォルダ名 |
| :--- | :--- |
| ドライブ | C: (デフォルトのシステムドライブ) |
| ユーザーディレクトリ | Users > [ユーザー名] |
| 隠しフォルダ | AppData > Local |
| ゲームフォルダ | PanicDelivery > Saved > SaveGames |

<Callout type="tip" title="プロからのヒント">
キーボードの `Win + R` を押し、`%localappdata%` と入力してEnterキーを押すことで、AppDataフォルダに素早くアクセスできます。これにより、非表示のシステムフォルダを手動で辿る手間が省けます。
</Callout>

<Section icon={Settings} title="進捗の管理とバックアップ">

特にパッチや修正が頻繁に行われるアーリーアクセス期間中は、定期的にセーブデータをバックアップしておくのが賢明です。Steamクラウドセーブは信頼性が高いですが、ローカルコピーを作成しておくことで、さらなる安全性が確保されます。

<CardGrid cols={2}>
  <Card title="手動バックアップ">
    - `SaveGames` フォルダ全体をコピーします。
    - セカンダリドライブやクラウドサービスに保存します。
    - バックアップには現在の日付（例：2026-06-22）を付けてラベル管理しましょう。
  </Card>
  <Card title="クラウド同期">
    - プロパティでSteamクラウドが有効になっていることを確認してください。
    - ライブラリで「最新の状態」であることを確認します。
    - ファイル同期中にSteamを中断させないようにしてください。
  </Card>
</CardGrid>

<Callout type="warning" title="重要な警告">
ゲームの実行中にセーブファイルの変更、削除、置換は絶対に行わないでください。これを行うと、ファイルが破損したり、キャラクターのカスタマイズや進捗が失われる可能性があります。セーブデータを操作する前には、必ずゲームを完全に終了させてください。
</Callout>

<Section icon={Target} title="セーブ問題のトラブルシューティング">

進捗が保存されない、またはゲームがプロファイルを読み込めないといった問題が発生した場合は、以下の手順に従ってファイルの整合性を確認してください。

<Steps>
  <Step num="1" title="ゲームファイルの整合性を確認">
    Steamを開き、Panic Deliveryを右クリックして「プロパティ」>「インストール済みファイル」を選択し、「ゲームファイルの整合性を確認」をクリックします。
  </Step>
  <Step num="2" title="権限を確認">
    ユーザーアカウントにAppDataフォルダへの読み取り/書き込み権限があることを確認してください。
  </Step>
  <Step num="3" title="キャッシュをクリア">
    Steamのダウンロードキャッシュをクリアすることで、ローカルマシンとクラウド間の同期の競合が解決する場合があります。
  </Step>
</Steps>

| 問題のタイプ | 推奨される解決策 |
| :--- | :--- |
| セーブが同期されない | プロパティでSteamクラウド設定を確認 |
| データが破損した | 手動のローカルバックアップから復元 |
| 進捗が消えた | Steamクライアントでゲームファイルの整合性を確認 |
| 権限エラー | Steamを管理者として実行 |

</Section>

<Section icon={FileText} title="データ構造の概要">

セーブフォルダには、構成ファイル、プレイヤー統計、コスメティックのアンロック情報が含まれています。各ファイルタイプが何を表しているかを理解することで、個人の進捗においてどのファイルが重要かを判断しやすくなります。

| ファイルタイプ | 説明 |
| :--- | :--- |
| `.sav` | 主要な進捗およびミッションデータ |
| `Config` | ユーザー設定およびグラフィック設定 |
| `Profiles` | プレイヤーIDおよびコスメティックインベントリ |
| `Logs` | クラッシュ報告用の技術データ |

<Callout type="success" title="成功">
ゲームを再インストールする場合や新しいコンピュータに移行する場合は、`.sav` ファイルを別のフォルダに保管しておいてください。これにより、苦労して手に入れたコスメティックやSpeetchy残高が確実に保護されます。
</Callout>

</Section>
