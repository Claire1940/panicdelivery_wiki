<Callout type="info" title="クイックガイド：Panic Delivery 修理アイテム">
- **Panic Deliveryの修理アイテム**やショップツールは、モンスターが蔓延る配送ルートを生き抜くために不可欠です。
- **Speetchy**は、ショップで防御装備やユーティリティアイテムを入手するための主要通貨です。
- チームで共有する「99の命」を守るため、トラッキングデバイスやモンスター忌避剤などの**生存ツールを優先**してください。
- ミッションの種類に応じて、常に誰かが適切なツールを装備しているよう**チームと連携**しましょう。
- **永続的なコスメティックアイテム**はランがリセットされても維持されますが、ミッション攻略に必要なアイテムは再購入が必要です。
</Callout>

<Section icon={Package} title="ショップアイテムと修理装備の理解">

Panic Deliveryでは、いかに効率的にリソースを管理できるかが生存の鍵となります。現在の早期アクセス版には15種類のユニークなアイテムが登場しますが、ユーティリティ、防御、ミッション特化型装備の違いを理解することが極めて重要です。修理アイテムやショップツールは単なる便利グッズではなく、配送の成功と、チームで共有する99の命を失うことの分かれ道となります。

<Callout type="tip" title="プロからのヒント">
Speetchyを使う前に、必ずミッション目標を確認しましょう。リカバリーミッションにおいて、使わないかもしれない即席武器を買うよりも、トラッキングデバイスに投資する方が遥かに価値があります。
</Callout>

ショップでは、アイテムはその主な機能に基づいて分類されています。以下は、これらのアイテムがランにどのような影響を与えるかの内訳です。

| アイテムカテゴリー | 主な機能 | 最適な用途 |
| :--- | :--- | :--- |
| **トラッキングデバイス** | 紛失した貨物の特定 | リカバリーミッション |
| **モンスター忌避剤** | 脱出経路の確保 | 高圧的な状況下での離脱 |
| **移動ツール** | 迅速な位置取り | 長く複雑なダンジョン |
| **防御ツール** | 近接戦闘の保護 | パニックファクトリー地帯 |

</Section>

<Section icon={Target} title="戦略的なアイテムロードアウト">

バランスの取れたチームのロードアウトを構築することは、手順化されたダンジョンレイアウトに対処するための最も信頼できる方法です。限られたクローンを共有しているため、チームメンバー全員が同じ種類の装備を持つ余裕はありません。

<Steps>
  <Step num="1" title="ミッションタイプを評価する">
    現在のランが「スタンダード」「リカバリー」「密輸」のどれであるかを確認します。これにより、移動速度が必要か、貨物探索のユーティリティが必要かが決まります。
  </Step>
  <Step num="2" title="役割を割り当てる">
    荷物を運ぶ「キャリア」と、バールや忌避剤などの防御装備を持つ「スカウト」を決めましょう。
  </Step>
  <Step num="3" title="Speetchyを割り振る">
    北極基地の探索トラップなど、現在のマップ特有のリスクを軽減するアイテムに通貨を使いましょう。
  </Step>
</Steps>

<Section icon={Shield} title="アイテムティアの概要">

プレイヤーの好みは分かれますが、チームの生存において客観的に高い価値を持つアイテムが存在します。以下の表は、現在のPanic Delivery早期アクセス版で利用可能なショップアイテムの有用性をまとめたものです。

| アイテム名 | ティア | 有用性 |
| :--- | :--- | :--- |
| **トラッキングデバイス** | S | リカバリーに不可欠 |
| **バール** | S | 高い汎用性 |
| **モンスター忌避剤** | A | 強力な脱出ユーティリティ |
| **移動ツール** | A | 優れた機動力 |
| **ガジェット** | B | 状況に応じたサポート |
| **グロアンチューブ** | B | 緊急時の防御 |

<Callout type="warning" title="リソース管理">
Speetchyを溜め込まないようにしましょう。命がゼロになるとランがリセットされるため、現在のミッションを完了するために役立つ装備にリソースを費やすことは、いつ来るかわからない「万が一」のために貯金するよりもほぼ確実に有益です。
</Callout>
