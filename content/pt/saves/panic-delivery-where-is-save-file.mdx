<Callout type="info" title="Guia Rápido">
- A dúvida sobre **onde ficam os arquivos de salvamento de Panic Delivery** refere-se à localização dos dados de salvamento locais no seu PC.
- A maioria dos títulos da Steam armazena o progresso na pasta **AppData** ou no diretório de instalação da **Steam**.
- Sempre use o **Steam Cloud** para evitar a perda de dados ao alternar entre diferentes estações de entrega.
- Recomenda-se realizar backups manuais antes de qualquer **verificação de integridade dos arquivos do jogo** ou atualizações do sistema.
</Callout>

<Section icon={Map} title="Localizando seus dados de salvamento">

Entender como seu progresso é armazenado é essencial para todo entregador nas zonas distópicas. Embora *Panic Delivery* gerencie a maior parte disso via Steam Cloud, saber a localização física do seu arquivo de salvamento é vital para solução de problemas, backups manuais ou migração de dados para uma nova máquina.

**Destaques do vídeo:**
- Entendendo o ciclo principal do progresso de entregador.
- Como as vidas compartilhadas impactam os dados da sua sessão local.
- Por que fazer backup do seu perfil é importante durante o Acesso Antecipado.

<Callout type="tip" title="Dica Profissional">
Se você não conseguir encontrar seus arquivos, certifique-se de que "Itens ocultos" esteja ativado nas configurações de exibição do Explorador de Arquivos do Windows, pois a pasta AppData é oculta por padrão.
</Callout>

| Tipo de Dado | Caminho Padrão | Importância |
| :--- | :--- | :--- |
| **Salvamento Local** | `%LOCALAPPDATA%\PanicDelivery\Saved` | **Crítico** |
| **Arquivos de Configuração** | `%LOCALAPPDATA%\PanicDelivery\Saved\Config` | Médio |
| **Steam Cloud** | `Steam\userdata\[IDdoUsuário]\2887450` | **Alta** |

</Section>

<Section icon={Target} title="Guia de Acesso Passo a Passo">

<Steps>
  <Step num="1" title="Abra o comando Executar">
    Pressione `Win + R` no seu teclado para abrir a caixa de diálogo Executar.
  </Step>
  <Step num="2" title="Navegue até AppData">
    Digite `%LOCALAPPDATA%` e pressione Enter. Isso abrirá a pasta de dados de aplicativos locais do seu perfil de usuário.
  </Step>
  <Step num="3" title="Localize a pasta do jogo">
    Procure pela pasta chamada `PanicDelivery`. Se ela não estiver presente, certifique-se de ter iniciado o jogo pelo menos uma vez.
  </Step>
  <Step num="4" title="Acesse os dados de salvamento">
    Navegue até `Saved` > `SaveGames`. É aqui que seu progresso de carreira e desbloqueios cosméticos são armazenados.
  </Step>
</Steps>

<Callout type="warning" title="Aviso de Segurança de Dados">
Nunca modifique esses arquivos enquanto o jogo estiver em execução. Editar seu arquivo de salvamento pode levar a um progresso corrompido, que pode não ser recuperável mesmo com a sincronização do Steam Cloud.
</Callout>

</Section>

<Section icon={Shield} title="Gerenciando a Sincronização na Nuvem">

O Steam Cloud é sua principal defesa contra a perda do seu progresso de entregador. Como *Panic Delivery* é um título em Acesso Antecipado, atualizações frequentes podem ocasionalmente causar conflitos de sincronização.

<CardGrid cols={2}>
  <Card title="Sincronização Steam Cloud">
    - Envia automaticamente seu progresso para os servidores da Valve.
    - Garante que seus dados acompanhem você em diferentes PCs.
    - Pode ser ativado ou desativado no menu de propriedades do jogo na Steam.
  </Card>
  <Card title="Backup Manual">
    - Copie a pasta `SaveGames` para uma unidade externa.
    - Útil antes de testar branches beta ou novos patches.
    - Oferece uma rede de segurança caso a sincronização na nuvem falhe.
  </Card>
</CardGrid>

| Cenário | Ação Recomendada |
| :--- | :--- |
| **Atualizando o Jogo** | Verifique a integridade via Steam, não toque nos salvamentos |
| **Trocando de PC** | Certifique-se de que o Steam Cloud esteja ativado |
| **Progresso Corrompido** | Restaure a partir do seu backup manual mais recente |
| **Novo Perfil de Entregador** | Exclua os arquivos locais (Use com cautela) |

</Section>
